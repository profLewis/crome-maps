Copernicus HRL Crop Types

Executive Summary

Pan-European crop type classification at 10m resolution from Sentinel-2, covering EEA-39 countries from 2017-2023. Part of the Copernicus Land Monitoring Service High Resolution Layer (HRL) Croplands product. Provides annual maps distinguishing major crop groups: cereals, root crops, industrial crops, pulses, fodder, vegetables, permanent crops, and grassland.

CoverageEurope (EEA-39)
Resolution10m (Sentinel-2)
Temporal2017-2023 (annual)
LicenseCopernicus Data Policy (free, open)

Description

The HRL Crop Types layer classifies agricultural land across Europe into major crop type groups using Sentinel-2 satellite imagery at 10m spatial resolution. It is produced by GeoVille for the Copernicus Land Monitoring Service (CLMS). The classification uses machine learning trained on LPIS (Land Parcel Identification System) reference data from EU member states.

Provenance

Produced by GeoVille GmbH under contract to the European Environment Agency (EEA) as part of the Copernicus Land Monitoring Service. Input data: Sentinel-2 L2A surface reflectance, supplemented by LPIS training data. Processing: annual classification using ensemble machine learning.

Classification System

Major crop groups: Cereals, Root Crops, Non-permanent Industrial, Dry Pulses/Protein, Fodder, Vegetables & Flowers, Permanent Crops, Grassland

Accuracy Assessment

Overall accuracy varies by year and region. Typical overall accuracy ~80-85% for major crop groups. Accuracy is higher for dominant crops (cereals, grassland) and lower for minor crops. See technical documentation for detailed per-class accuracy metrics.
